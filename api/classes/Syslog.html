    Class Syslog

    Main Syslog class providing fluent interface for system logging

    This class wraps the native syslog(3) system calls with a modern TypeScript API. All methods are synchronous and return the instance for method chaining.

    Example

    const logger = new Syslog({
      ident: 'my-app',
      facility: SyslogFacility.LOG_LOCAL0
    });

    logger.info('Application started')
          .error('Something went wrong', { code: 500 });

    • new Syslog(options?: SyslogOptions): Syslog

      Create a new Syslog instance

      Initializes the syslog connection with the provided options. If no options are provided, sensible defaults are used.

      Parameters

      • options: SyslogOptions = {}

        Configuration options for the syslog connection

      Returns Syslog

      Example

      // Default configuration
      const logger = new Syslog();

      // Custom configuration
      const logger = new Syslog({
        ident: 'web-server',
        facility: SyslogFacility.LOG_DAEMON,
        options: SyslogOption.LOG_PID | SyslogOption.LOG_CONS
      });
